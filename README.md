### What is this project?
A very simple project/game to use and test some features of [MyEngine](https://github.com/LouisH96/MyEngine).

### How to run this project?
* Download the .zip from [Releases](https://github.com/LouisH96/Tower/releases/tag/v1.0.0) 
* Extract and run the .exe

### How to build this project?
* Clone [MyEngine](https://github.com/LouisH96/MyEngine)
* Create an environment variable named **MyEngine** that points to **_(YourPath)_\\MyEngine\\source\\** <br/>
  <sup>(this folder should include the folders: *MyBasics*, *MyEngine*, *MyFiles*)</sup>
* Clone & Build this project
